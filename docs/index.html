ゲームについて

このゲームは、住宅街で迷子になった4匹のペット（犬・猫・うさぎ・鳥）を3分以内に見つけて救出するアドベンチャーゲームです。

  • 制限時間：3分間
  • 4種類のペット（犬・猫・うさぎ・鳥）
  • リアルな住宅街マップ
  • 多言語対応（日本語・英語）
  • 時間経過でヒント表示

技術仕様

  • Python + Pygame製
  • Pygbag でWeb対応
  • 解像度：1280×720
  • 60FPS動作
  • 日本語フォント完全対応
