# ミステリー・ペット・レスキュー 🐾

## ゲーム概要
迷子になったペットたちを探し出して救出する、20-30分で楽しめるコンパクトなアドベンチャーゲームです。

### 主な特徴
- **探索**: タイルベースの住宅街マップを歩き回り、隠れているペットを発見
- **4種類のペット**: 犬・猫・うさぎ・鳥を救出
- **ミニゲーム**: アクションゲームと記憶ゲームでペットとの信頼関係を構築
- **謎解き**: アイテムの組み合わせで進行する簡単なパズル
- **多言語対応**: 日本語・英語に対応（予定）

### ゲームの流れ
1. 住宅街を探索して4匹のペットを発見
2. ミニゲームまたは謎解きでペットとの信頼関係を構築
3. ペットを救出
4. 全ペット救出でゲームクリア！

## 動作環境
- Python 3.8以上
- Pygame 2.0以上
- 画面解像度: 1280x720以上推奨

## 操作方法

### 基本操作
- **WASD / 矢印キー**: 移動（8方向）
- **Shift + 移動**: 走行（スタミナ消費）
- **E / Enter**: ペットとの相互作用
- **ESC**: 一時停止/メニュー
- **SPACE**: アクション/決定

### ゲーム中の操作
- **M**: ミニマップ表示切り替え
- **F**: フルスクリーン切り替え

### デバッグ・表示切り替え（開発用）
- **F1**: デバッグ表示切り替え
- **F2**: 統計表示切り替え  
- **F3**: グリッド表示切り替え
- **F4**: FPS表示切り替え
- **R**: シーンリセット

## 開発者向け

### プロジェクト構造
```
mystery-pet-rescue/
├── src/
│   ├── game_main.py     # メインゲームエントリーポイント
│   ├── game_flow.py     # ゲームフロー管理
│   ├── entities/        # プレイヤー、ペット、NPCクラス
│   ├── systems/         # ゲームシステム（インベントリ、セーブ等）
│   ├── scenes/          # 各画面（メニュー、ゲーム、結果）
│   ├── ui/              # UI要素（HUD、ダイアログ等）
│   └── utils/           # ユーティリティ（スプライト管理等）
├── assets/              # ゲームアセット
│   ├── images/          # スプライト画像
│   ├── sounds/          # BGM・効果音（未実装）
│   └── fonts/           # フォントファイル
├── data/                # ゲームデータ
│   ├── maps/            # マップデータ（JSON）
│   ├── pets/            # ペットデータ（JSON）
│   └── puzzles/         # パズルデータ（JSON）
├── config/              # 設定ファイル
├── saves/               # セーブデータ
├── tests/               # テストファイル
├── examples/            # デモ・サンプル実行ファイル
└── requirements.txt     # 依存関係
```

### 開発環境セットアップ
```bash
# 開発用依存関係のインストール
pip install -r requirements.txt

# テスト実行
python -m pytest tests/

# コード品質チェック
flake8 src/
black src/
```
