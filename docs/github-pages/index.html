🐾 ミステリー・ペットレスキュー

住宅街で迷子になったペットたちを救出しよう！

ゲーム概要

ミステリー・ペットレスキューは、住宅街で迷子になった4匹のペットを時間内に救出する2Dアドベンチャーゲームです。

プレイヤーは3分という制限時間の中で、広大な住宅街マップを探索し、隠れているペットたちを見つけ出さなければなりません。

🏘️

リアルな住宅街

本格的な都市マップで、建物や公園を探索

🐕

4種類のペット

犬・猫・うさぎ・鳥を救出

⏱️

時間制限チャレンジ

3分以内に全ペットを救出！

🌐

多言語対応

日本語・英語に完全対応

操作方法

WASD / 矢印キー

プレイヤーの移動

E キー

ペットの救出（近くにいる時）

P キー

ゲームの一時停止

ESC キー

メニューに戻る

F3 キー

FPS表示の切り替え

1～3 キー

セーブスロット選択

ゲームの特徴

  • 🎵 Beatoven.ai APIによる動的なBGM生成
  • 💾 3つのセーブスロットとオートセーブ機能
  • 🗺️ ミニマップで現在地を確認
  • 💡 時間経過でヒントが表示
  • 🏆 救出時間に応じたスコアシステム
  • 🎮 直感的な操作とスムーズなゲームプレイ

インストール方法

Windows版

  1. 上のボタンからWindows版をダウンロード
  2. ダウンロードしたZIPファイルを解凍
  3. 解凍したフォルダ内の「MysteryPetRescue.exe」をダブルクリックして実行

Mac版

  1. 上のボタンからMac版をダウンロード
  2. ダウンロードしたZIPファイルを解凍
  3. 解凍したアプリケーションを「アプリケーション」フォルダに移動
  4. 初回起動時は右クリックして「開く」を選択
