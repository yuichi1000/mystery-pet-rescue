# プロジェクト概要

## プロジェクト名
ミステリー・ペット・レスキュー (Mystery Pet Rescue)

## 概要
10-14歳の子供たちを対象とした、行方不明のペットを探し出すアドベンチャーゲーム。
プレイヤーは探偵となり、様々なエリアで手がかりを集め、パズルを解き、ペットを救出する。

## ターゲット
- 対象年齢：10-14歳
- プレイ時間：v1.0は1-2時間（住宅街エリアのみ）
- 難易度：年齢に応じた段階的な難易度設定

## 技術スタック
- 言語：Python 3.9+
- ゲームエンジン：Pygame 2.5+
- UI フレームワーク：pygame-gui
- セーブシステム：JSON + 暗号化（cryptography）
- 音楽生成：Beatoven.ai API
- テスト：pytest + Playwright（MCP統合）
- 多言語対応：gettext

## リリース戦略
### v1.0（初期リリース）
- 住宅街エリアのみ
- 基本的なゲームメカニクス
- 日本語・英語対応
- 1-2時間のゲームプレイ

### v2.0（将来計画）

### v3.0（将来計画）

## 主要機能
1. **探索システム**
   - ポイント＆クリック操作
   - インタラクティブオブジェクト
   - 手がかり収集

2. **パズルシステム**
   - 論理パズル
   - ミニゲーム
   - 暗号解読

3. **対話システム**
   - NPCとの会話
   - 選択肢による分岐
   - 手がかりの獲得

4. **インベントリシステム**
   - アイテム管理
   - アイテム組み合わせ
   - ヒントシステム

5. **セーブシステム**
   - 自動セーブ
   - 手動セーブ（3スロット）
   - 暗号化による改ざん防止

6. **音楽システム**
   - Beatoven.ai による動的BGM生成
   - シーンに応じた音楽切り替え
   - 効果音管理

## アクセシビリティ
- カラーブラインド対応
- テキストサイズ調整
- キーボード完全対応
- 音声なしでもプレイ可能
